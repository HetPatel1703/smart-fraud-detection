# Smart Credit Card Fraud Investigation

A complete fraud detection pipeline with SHAP‑based explanations, a continuous risk score, and an investigator dashboard.

## Dataset
- **Name**: Kaggle Credit Card Fraud Detection  
- **Source**: [https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud)  
- **License**: Open (CC0)  
- **Preprocessing**: None applied; raw PCA features used.
